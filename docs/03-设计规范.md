# 设计规范 - Diary App

## 设计理念

- **简洁直观**：界面清爽，操作一目了然
- **专注内容**：减少视觉干扰，突出日记内容
- **温和舒适**：淡蓝色主题营造轻松氛围

---

## 色彩系统

### 主色调 - 淡蓝色系

| 名称 | 色值 | 用途 |
|------|------|------|
| 主色 | `#6BA3E0` | 按钮、链接、选中状态 |
| 主色浅 | `#E8F4FD` | 背景、卡片底色 |
| 主色深 | `#4A90D9` | 悬停状态、强调 |
| 主色淡 | `#F0F7FC` | 页面背景 |

### 辅助色

| 名称 | 色值 | 用途 |
|------|------|------|
| 文字主色 | `#333333` | 正文、标题 |
| 文字次色 | `#666666` | 辅助文字、时间 |
| 文字浅色 | `#999999` | 占位符、禁用 |
| 边框色 | `#E0E0E0` | 分割线、边框 |
| 白色 | `#FFFFFF` | 卡片背景、输入框 |

| 危险色 | `#FF6B6B` | 删除按钮 |

## 间距系统

### 基础单位：8px

| 名称 | 数值 | 用途 |
|------|------|------|
| xs | 4px | 紧密间距 |
| sm | 8px | 小间距 |
| md | 16px | 标准间距 |
| lg | 24px | 大间距 |
| xl | 32px | 区块间距 |

### 常见用法

- 页面内边距：`16px` (md)
- 卡片内边距：`16px` (md)
- 元素间距：`8px` (sm) 或 `16px` (md)
- 区块间距：`24px` (lg)

## 组件样式

### 按钮

**主按钮**
```css
background: #6BA3E0;
color: #FFFFFF;
border: none;
border-radius: 8px;
padding: 10px 20px;
font-size: 14px;
cursor: pointer;
```

**次按钮**
```css
background: #FFFFFF;
color: #6BA3E0;
border: 1px solid #6BA3E0;
```

**危险按钮**
```css
background: #FF6B6B;
color: #FFFFFF;
```

### 卡片（日记条目）

```css
background: #FFFFFF;
border-radius: 12px;
padding: 16px;
box-shadow: 0 2px 8px rgba(0, 0, 0, 0.08);
```

**置顶卡片特殊样式**
```css
border-left: 4px solid #FFD93D;
```

### 输入框

```css
background: #FFFFFF;
border: 1px solid #E0E0E0;
border-radius: 6px;
padding: 10px 12px;
font-size: 14px;
```

**聚焦状态**
```css
border-color: #6BA3E0;
outline: none;
box-shadow: 0 0 0 3px rgba(107, 163, 224, 0.2);
```

### 日历组件

- 整体圆角：`12px`
- 日期格子：`36px x 36px`
- 有日记标记：底部小圆点，颜色 `#FFB366`
- 选中日期：背景 `#6BA3E0`，文字白色
- 今天标记：边框 `#6BA3E0`

## 布局结构

### 页面整体布局

```
┌─────────────────────────────────────┐
│  顶部栏 (Logo + 导出/导入按钮)        │
├─────────────────────────────────────┤
│  搜索框                              │
├─────────────────────────────────────┤
│  ┌──────────┬───────────────────┐   │
│  │          │                   │   │
│  │  日历    │   日记列表         │   │
│  │  组件    │   + 写日记按钮     │   │
│  │          │                   │   │
│  └──────────┴───────────────────┘   │
└─────────────────────────────────────┘
```

### 响应式断点

| 断点 | 宽度 | 布局变化 |
|------|------|----------|
| 桌面 | > 768px | 左右布局，日历在左，列表在右 |
| 平板/手机 | ≤ 768px | 上下布局，日历在上，列表在下 |

## 交互反馈

### 悬停效果
- 按钮：背景色变深 `10%`
- 卡片：阴影加深，轻微上移 `2px`
- 链接：显示下划线

### 点击效果
- 按钮：轻微缩放 `scale(0.98)`
- 卡片：阴影内收

### 过渡动画
```css
transition: all 0.2s ease;
```

## 图标规范

使用 SVG 图标，大小：
- 小图标：`16px`（如删除、编辑）
- 中图标：`20px`（如搜索、日历）
- 大图标：`24px`（如空状态插图）

建议图标：
- 搜索：`search`
- 日历：`calendar`
- 编辑：`edit-2`
- 删除：`trash-2`
- 置顶：`pin`
- 导出：`download`
- 导入：`upload`
- 添加：`plus`
