# 需求文档 - Diary App

## 项目概述

一个纯前端的本地日记应用，用户双击 HTML 文件即可在浏览器中打开使用，无需安装，数据存储在本地。

---

## 功能需求

### 核心功能

| 功能 | 优先级 | 说明 |
|------|--------|------|
| 写日记 | P0 | 支持文字内容，可插入图片 |
| 编辑日记 | P0 | 发布后可以随时修改内容 |
| 删除日记 | P0 | 可以删除不需要的日记 |
| 置顶日记 | P1 | 某篇日记可以置顶，优先显示在列表顶部 |
| 日历视图 | P1 | 显示日历，有日记的日期做标记，点击日期筛选查看 |
| 搜索功能 | P1 | 按关键词搜索日记标题和内容 |
| 列表排序 | P0 | 按时间从新到旧排列 |
| 数据导出 | P1 | 将所有日记导出为 JSON 文件备份到本地 |
| 数据导入 | P1 | 从导出的 JSON 文件恢复数据 |

### 详细功能描述

#### 写日记
- 支持多行文本输入
- 可以插入本地图片（支持拖拽和选择文件）
- 图片自动压缩存储
- 自动记录创建时间
- 保存时可选择是否置顶

#### 编辑日记
- 点击已有日记进入编辑模式
- 可以修改文字内容
- 可以增删图片
- 保存后更新时间戳

#### 删除日记
- 每篇日记有删除按钮
- 删除前需要确认，防止误删

#### 置顶功能
- 置顶日记显示在列表最上方
- 多篇置顶日记按置顶时间排序
- 可以取消置顶

#### 日历视图
- 显示当月日历
- 有日记的日期显示标记（圆点或高亮）
- 点击日期筛选显示该日日记
- 可以切换月份查看

#### 搜索功能
- 搜索框输入关键词
- 实时搜索日记标题和内容
- 搜索结果按时间排序

#### 数据备份
- 导出：生成 JSON 文件下载到本地
- 导入：选择之前导出的 JSON 文件恢复
- 导入时可选择合并或覆盖现有数据

---

## 非功能需求

### 性能要求
- 页面加载时间 < 2 秒
- 搜索响应时间 < 300ms
- 支持存储 1000+ 篇日记不卡顿

### 兼容性要求
- 支持 Chrome、Edge、Firefox 最新版本
- 支持 Windows 10/11 系统

### 数据安全
- 数据本地存储，不上传到任何服务器
- 定期提醒用户导出备份
