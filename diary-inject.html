📝 日记注入工具

由于浏览器的同源策略限制，本地文件之间无法共享存储数据。请按以下步骤在 diary-app 页面内完成注入。

为什么需要这样做？
浏览器将每个本地 HTML 文件的存储空间完全隔离。diary-inject.html 写入的数据，index.html 无法读取。因此注入操作必须在 diary-app 自身的页面上下文中执行。
  1. 打开 diary-app
    在浏览器中打开 index.html
  2. 打开开发者控制台
    按 F12（或 Ctrl+Shift+J / Cmd+Option+J）打开开发者工具，切换到 Console 标签
  3. 执行注入命令
    在控制台粘贴以下命令并按回车：
    injectTodayDiary()
  4. 查看结果
    注入成功后页面会自动刷新日记列表，今天的日记将置顶显示

备选方案：通过导入功能

如果控制台方式不可用，也可以：

  1. 在 diary-app 中点击「设置」→「导出」，先导出一份现有数据作为备份
  2. 将导出的 JSON 中的 entries 数组追加今天的日记数据（格式见上方预览）
  3. 使用「导入」功能重新导入合并后的数据
