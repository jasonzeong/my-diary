# Diary App - CLAUDE.md

## 项目概述

一个纯前端的本地日记应用，双击 `index.html` 即可在浏览器中打开使用。

**核心特点**：
- 无需安装，纯 HTML/CSS/JS 实现
- 支持文字 + 图片日记
- 本地数据存储（IndexedDB）
- 支持导出/导入备份

## 常用操作

### 打开应用
直接双击 `index.html` 文件，在浏览器中打开即可。

### 开发调试
在浏览器中按 `F12` 打开开发者工具，可以查看控制台日志和调试代码。

### 数据备份
使用应用内的「导出」按钮，将数据保存为 JSON 文件到本地。

---

## 注意事项

1. **浏览器兼容性**：建议使用 Chrome 或 Edge 浏览器
2. **数据存储**：数据存储在浏览器 IndexedDB 中，清理浏览器数据会丢失
3. **定期备份**：建议定期使用导出功能备份数据
4. **图片大小**：单张图片会自动压缩，但建议不要上传过大的原图
