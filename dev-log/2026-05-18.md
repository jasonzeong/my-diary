## Bug 修复记录

### Bug 1：弹窗无法显示
**现象**：写日记和导入弹窗无法打开  
**原因**：CSS 使用 `.active` 类控制显示，但 JS 还在用 `style.display`  
**解决**：统一改用 `classList.add/remove('active')`

### Bug 2：导入失败
**现象**：点击导入按钮后报错  
**原因**：`index.html` 缺少 `<div id="toast">` 元素  
**解决**：在 HTML 中添加 Toast 容器节点

### Bug 3：边界情况处理
**现象**：某些无效数据可能导致页面崩溃  
**原因**：缺少对无效日期、null 值的防御性处理  
**解决**：
- `filterByDate`：添加无效日期检测
- `handleSearch`：处理 title/content 可能为 null 的情况
- `renderEntry`：处理无效日期显示为"未知日期"
